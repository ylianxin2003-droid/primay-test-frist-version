# 云端 API-only 运行说明

当前版本只读取 SERENE 在线数据，不读取本地样例文件，也不会在 API 失败时偷偷显示演示数据。

## Streamlit Cloud Secrets

在应用的 **Settings -> Secrets** 中填写：

```toml
SERENE_API_BASE_URL = "https://serene.bham.ac.uk"
SERENE_API_TOKEN = "你的 API token"
SERENE_API_TIMEOUT = "30"
SERENE_AUTH_SCHEME = "Token"
```

保存后重启应用。浏览器里已经登录 SERENE 并不能让 Streamlit Cloud 自动登录；云端必须使用有 AIDA 输出读取权限的 API token。若 token 被输出页面拒绝，页面会明确提示返回了登录页。

## 当前读取方式

1. 用 token 读取 AIDA 输出目录。
2. 为每个选中时间寻找最近的 `param_2d` 产品。
3. 每个不同输出时间只下载一次 HDF5。
4. 在内存中按照用户指定的边界框和经纬度间距插值取点。
5. 单独读取全球 Kp/ap，并只作为背景指标展示。

因此，改变地图范围或把间距调密只会改变本地地图点数，不会产生逐点 API 请求，也不会自动修改用户选择的间距。历史事件快捷选项从 2024 年开始。

## 验证

- 侧边栏显示 `Data source: SERENE API` 和 `Verified model: AIDA`。
- 状态区分别显示 `AIDA datasets downloaded` 与 `Local map points`。
- 全球 Kp/ap 有独立面板，不出现在区域风险地图中。
- API 失败时显示错误和警告，不显示本地替代数据。
