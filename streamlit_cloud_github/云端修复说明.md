# 云端 API-only 运行说明

当前版本使用 SERENE 官方 raw-output API：

```text
https://spaceweather.bham.ac.uk/api/download-output/
```

每个不同输出时间只下载一次 raw AIDA HDF5，然后使用导师 Benjamin Reid
公开的 [`breid-phys/aida-ionosphere`](https://github.com/breid-phys/aida-ionosphere)
v0.1.3 在 Streamlit 内存中按边界框和间距计算地图。相关接口和计算边界均在
代码旁标注来源，未复制导师的科学模型实现。

## 必须重新部署

导师包要求 `pandas<2`、`numpy<2`。Streamlit Cloud 需要选择 Python 3.11，
而已有应用不能直接修改 Python 版本：

1. 记录当前应用 URL 和 Secrets。
2. 删除当前 Streamlit 应用。
3. 重新创建，入口选择 `streamlit_cloud_github/app.py`。
4. 在 Advanced settings 中选择 Python 3.11。
5. 填入新 Secrets 后部署。

```toml
SERENE_API_BASE_URL = "https://spaceweather.bham.ac.uk"
SERENE_API_TOKEN = "新生成的 token"
SERENE_API_TIMEOUT = "30"
SERENE_AUTH_SCHEME = "Token"
```

之前发到聊天中的 token 已暴露，必须撤销，不能继续使用。

## 正确结果

- API 测试显示 `Connected to SERENE AIDA raw-output API`。
- 状态区分别显示 raw 数据集下载次数和本地地图点数。
- 改变网格间距不会改变相同时间对应的下载次数。
- Kp/ap 只显示在全球指标面板。
- API 失败时不显示本地假数据。
