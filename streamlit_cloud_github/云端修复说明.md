1. **SERENE API 未配置**  
   Streamlit Cloud **不会**读取你电脑上的 `.env`，必须在网页 **Settings → Secrets** 里填写 token。

2. **本地样例数据已移除**  
   当前版本只通过在线 SERENE API/resource call 获取数据，不再读取 `data/*.json` 或本地回退数据。

## Streamlit Cloud 配置 Secrets

1. 打开 https://share.streamlit.io → 你的应用 `2026-project`
2. **Settings** → **Secrets**
3. 粘贴：

```toml
SERENE_API_BASE_URL = "https://spaceweather.bham.ac.uk"
SERENE_API_TOKEN = "你的token"
SERENE_API_TIMEOUT = "30"
SERENE_AUTH_SCHEME = "Token"
```

4. **Save** → **Reboot app**
