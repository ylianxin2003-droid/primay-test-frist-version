# GitHub + Streamlit Cloud 部署指南

本文件夹 **`streamlit_cloud_github`** 可直接作为 **Git 仓库根目录** 推送到 GitHub，并在 Streamlit Cloud 上免费发布为公开网站。

---

## 一、文件夹里有什么

| 文件 | 作用 |
|------|------|
| `app.py` | 网站入口（Streamlit 主程序） |
| `config.py` | 读取本地 `.env` 或云端 Secrets |
| `serene_client.py` | SERENE API（官方 `Token` + `/api/calc/`） |
| `data_loader.py` | 在线 SERENE API 数据加载 |
| `alert_engine.py` | 原型 ICAO-style 告警 |
| `visualisation.py` | 图表 |
| `requirements.txt` | 依赖列表 |
| `.streamlit/config.toml` | Streamlit 配置 |
| `.streamlit/secrets.toml.example` | 云端密钥模板（仅供参考） |

**不要上传：** `.env`、`.streamlit/secrets.toml`（含真实 token）

---

## 二、上传到 GitHub

### 方法 A：网页上传

1. GitHub 新建仓库（Public）
2. **Upload files**，把本文件夹内**所有文件**拖进去（保持目录结构）
3. Commit

### 方法 B：命令行

```powershell
cd "你的路径\streamlit_cloud_github"
git init
git add .
git commit -m "Aviation space weather dashboard"
git branch -M main
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

---

## 三、Streamlit Cloud 部署

1. 打开 https://share.streamlit.io ，用 GitHub 登录  
2. **Create app** → 选择刚建的仓库  
3. **Main file path** 填：`app.py`  
4. **Branch**：`main`  
5. 展开 **Advanced settings** → **Secrets**，粘贴：

```toml
SERENE_API_BASE_URL = "https://spaceweather.bham.ac.uk"
SERENE_API_TOKEN = "你的token"
SERENE_API_TIMEOUT = "30"
SERENE_AUTH_SCHEME = "Token"
```

6. 点击 **Deploy**  
7. 等待 1–3 分钟，获得链接，例如：  
   `https://aviation-space-weather-xxxx.streamlit.app`

把该链接写进论文或答辩 PPT 即可。

---

## 四、使用与安全建议

1. 本应用只通过在线 SERENE API/resource call 获取数据，不再读取或提交本地样例数据  
2. **不要**把 token 写进代码或提交到 GitHub  
3. 告警仅为 **prototype advisories**，需在页面注明非官方 ICAO 预警  

---

## 五、常见问题

| 问题 | 处理 |
|------|------|
| 部署失败、找不到模块 | 确认 `requirements.txt` 在仓库根目录 |
| API 不工作 | 检查 Streamlit Cloud → Settings → Secrets 是否填写正确 |
| 页面一直 loading | API 区域过大；用小范围 + 大 grid step，确认 API call 估算不超过 50 |
| 更新代码后网站没变 | Streamlit Cloud 一般会自动重建；可在控制台点 **Reboot app** |

## 六、官方 SERENE API 格式

```bash
curl -X POST \
  -H "Authorization: Token <token>" \
  -d latitude=52.4862 -d longitude=1.8904 \
  https://spaceweather.bham.ac.uk/api/calc/
```
